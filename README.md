# Rock-Paper-Scissors Game

This is a simple **Rock-Paper-Scissors** game built using **HTML**, **CSS**, and **JavaScript**. The game allows users to play against the computer for a total of 20 rounds, after which the winner is declared based on the final scores.

## Table of Contents
- [Features](#features)
- [Technologies Used](#technologies-used)
- [How to Play](#how-to-play)
- [Setup Instructions](#setup-instructions)


## Features
- **User vs. Computer Gameplay**: Players can choose between Rock, Paper, or Scissors.
- **20 Rounds Limit**: The game automatically ends after 20 rounds, and the winner is declared based on the highest score.
- **Responsive Design**: The game works well on both desktop and mobile devices.
- **Interactive User Interface**: The choices (Rock, Paper, Scissors) have a hover effect, and the score updates in real-time.



## Technologies Used
- **HTML5**: For structuring the webpage.
- **CSS3**: For styling and layout.
- **JavaScript**: For game logic and interactivity.

## How to Play
1. **Choose Rock, Paper, or Scissors**: Click on one of the options to make your move.
2. **See the Computer's Move**: The computer will randomly choose its option.
3. **Score Update**: The score will be updated based on the result (win, lose, or draw).
4. **20 Rounds**: The game lasts for 20 rounds. After 20 rounds, the player with the higher score wins.
5. **Final Result**: The final winner will be declared after all 20 rounds are played.
